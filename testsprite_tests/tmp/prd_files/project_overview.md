# AI Finance Ops Copilot

A financial operations dashboard that aggregates data from multiple payment providers (Stripe, Lemon Squeezy, PayPal) and bank connections (Plaid). Provides MRR tracking, cash flow forecasting, analytics, and invoice management.

## Target Users
- SaaS founders and finance teams
- Stripe/Lemon Squeezy/PayPal merchants
- Businesses needing consolidated financial data

## Core Features
1. Multi-provider revenue aggregation (Stripe, Lemon Squeezy, PayPal)
2. Bank account connectivity via Plaid
3. MRR and cash flow analytics
4. Invoice management
5. Subscription tracking
6. Webhook processing for real-time updates
