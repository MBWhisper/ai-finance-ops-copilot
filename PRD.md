# AI Finance Ops - Product Requirements

## Overview
SaaS financial analytics dashboard for founders.
Connects Stripe, PayPal, Lemon Squeezy, and bank 
accounts via Plaid to track MRR, cash flow, and runway.

## Key Pages
- / → Marketing homepage
- /login → Authentication
- /signup → Registration
- /dashboard/overview → KPI metrics
- /dashboard/analytics → Charts and trends
- /dashboard/cashflow → Cash flow forecasting
- /dashboard/settings → Integration management

## Core Features
- Real-time MRR tracking
- 90-day cash flow forecast
- Multi-platform revenue aggregation
- Bank account connection via Plaid
- PayPal Business integration
- Lemon Squeezy subscription tracking
